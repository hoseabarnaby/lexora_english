# Lexora English Boost v24 — No Login

Versi ini langsung membuka Dashboard.

## Tidak ada
- Login atau pembuatan akun
- QRIS / premium
- Admin approval
- Supabase

## Penyimpanan progress
Progress belajar disimpan otomatis menggunakan `localStorage` browser, termasuk:
- flashcard review
- hard words
- quiz scores
- completed modules
- streak
- grammar progress
- writing dan speaking notes

Karena tidak memakai akun, progress hanya tersedia di browser/perangkat yang sama. Jangan menghapus data browser jika ingin mempertahankan progress.

## Menjalankan website

```bash
python -m http.server 5500
```

Lalu buka `http://localhost:5500`.

Bisa juga memakai VS Code Live Server atau langsung di-host di Vercel, Netlify, dan GitHub Pages.
